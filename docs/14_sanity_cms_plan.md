# Sanity CMS Plan

Sanity should be used as the content management panel.

Admin URL can be:

```txt
/studio
```

or external Sanity Studio deployment.

---

## CMS sections

1. Products
2. FEFCO Types
3. Gallery
4. Pages
5. Site Settings

---

## Who edits what

Client/admin can edit:

• product texts
• product images
• FEFCO descriptions
• gallery images
• contact details
• SEO titles and descriptions

Developer controls:

• layout
• components
• logic
• form handling
• deployment

---

## Recommended MVP CMS setup

For first version, CMS should include:

Products
FEFCO Types
Site Settings

Gallery and editable Pages can be added after MVP if needed.
