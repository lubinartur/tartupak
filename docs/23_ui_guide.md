
# 23 UI Guide — Tartupak

## Design Style

Scandinavian industrial minimalism with kraft cardboard accents.

Visual direction:
- clean
- industrial
- minimal
- B2B manufacturing aesthetic
- large spacing
- strong product visuals

Avoid:
- SaaS gradients
- dark UI themes
- ecommerce-style product grids

---

## Color Palette

Primary (industrial green)
#1F3D2B

Accent (kraft cardboard)
#C79A5B

Background
#FFFFFF

Secondary background
#F4EFEA

Text color
#2B2B2B

---

## Typography

Headings:
Geist or Inter

Body:
Inter

Guidelines:
- large headings
- short paragraphs
- generous line spacing

---

## Layout Principles

Use a spacious grid layout.

Section order on homepage:

Hero  
Product Categories  
Custom Packaging  
Manufacturing Process  
FEFCO Preview  
Why Tartupak  
Contact Section

Each section should have strong vertical spacing.

Containers:
max width ~1200px

---

## Buttons

Primary button:
dark green background
white text

Example:
Request quote

Secondary button:
outlined style

Example:
View FEFCO catalog

Hover:
slightly darker green

---

## Cards

### Product Card

Fields:
- image
- title

- CTA button

Layout:
vertical card
large image
minimal text

---

### FEFCO Card

Fields:
- FEFCO code
- illustration
- description
- usage

Style:
technical / simple

---

### Feature Card

Used for:
Why Tartupak

Fields:
- icon
- title
- short explanation

---

## Contact Form

Fields:

Required:
- name
- email
- message

Optional:
- company
- phone
- packaging type
- dimensions
- quantity
- file upload

Layout:
clean industrial form
2-column desktop layout

---

## Imagery

Preferred visuals:

- cardboard sheets
- packaging production machines
- corrugated cardboard
- packaging prototypes
- warehouse packaging

Avoid:
stocky corporate imagery
generic office scenes

---

## Spacing

Use generous whitespace.

Typical section spacing:
80px–120px vertical.

---

## Icon Style

Simple line icons.
Industrial neutral tone.

---

## Overall Impression

The website should feel:

reliable  
industrial  
clean  
professional  

Goal:
clear B2B packaging manufacturer website focused on generating quote requests.
