# AI Context for Cursor

This repository contains the Tartupak website project.

Tartupak is a B2B cardboard and corrugated packaging manufacturer website.

The project goal is to create a modern, clean, industrial website focused on lead generation.

Documentation is located in `/docs`.

Cursor should always follow the documentation files before generating or changing code.

---

## Core direction

Build a modern B2B packaging manufacturer website.

Main priorities:

1. Clear product presentation
2. Easy quote request flow
3. FEFCO catalog reference
4. Trustworthy industrial design
5. Simple CMS editing
