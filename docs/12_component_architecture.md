# Component Architecture

## Layout components

### Header
Purpose:
Main navigation and contact CTA.

Includes:
• logo
• navigation links
• language switcher placeholder
• quote request button

### Footer
Purpose:
Company info, contact details, navigation and legal links.

### MobileNav
Purpose:
Mobile navigation drawer.

---

## Section components

### Hero
Purpose:
First impression and main CTA.

Content:
• headline
• subheadline
• primary CTA
• secondary CTA
• product/manufacturing visual

### ProductCategories
Purpose:
Show main packaging categories.

### CustomPackaging
Purpose:
Explain that Tartupak can produce packaging by custom dimensions, quantity and purpose.

### ManufacturingProcess
Purpose:
Show simple process:
1. Request
2. Measurements
3. Sample / confirmation
4. Production
5. Delivery

### FefcoPreview
Purpose:
Preview several FEFCO types and link to full catalog.

### WhyTartupak
Purpose:
Trust-building section.

Possible points:
• local manufacturer
• flexible quantities
• custom production
• fast communication
• B2B experience

### ContactSection
Purpose:
Lead generation form.

---
