# Project Overview

## Company

**Tartu Pak OÜ** — Estonian corrugated cardboard packaging manufacturer.
Founded: 2001
Location: Tamme 35, Tõrvandi alevik, Ülenurme vald, 61715 Tartumaa, Estonia
Credit rating: AA
Recognition: Edukas Eesti Ettevõte (multiple years)

---

## Goal

Create a modern B2B website that:
- Presents packaging products clearly
- Provides interactive FEFCO catalogue reference
- Allows clients to request custom packaging quotes
- Generates B2B leads via contact form

**Primary focus: lead generation.**

---

## Target Audience

- Logistics and freight companies
- Food manufacturers and restaurants
- E-commerce businesses
- Warehouses and distributors
- Industrial companies
- Retail brands needing printed packaging

---

## Key Features

- Interactive FEFCO catalog with search + series filter
- Quote request form with pre-fill from product/FEFCO pages
- Multilingual (ET/RU/EN)
- Mobile responsive
- SEO optimized (Next.js SSR/SSG)
- CMS-managed content via Sanity
