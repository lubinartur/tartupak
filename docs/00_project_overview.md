# Tartupak Website

## Goal
Create a modern B2B website for Tartupak — a cardboard packaging manufacturer.

The website should:
• Present packaging products
• Provide FEFCO catalogue reference
• Allow clients to request custom packaging
• Generate B2B leads via contact form

Primary focus: lead generation.

---

## Competitors
https://pakendid.ee
https://cista.ee

Weaknesses of competitors:
• outdated design
• poor UX
• slow navigation
• heavy catalog structure

Tartupak goal:
• cleaner UX
• modern industrial design
• easier quote requests
