# FEFCO Catalog Structure

FEFCO is the international standard for corrugated packaging designs.

The website should present commonly used FEFCO types.

Example entries:

FEFCO 0201
Standard shipping box.

FEFCO 0203
Overlap slotted container.

FEFCO 0210
Folder type box.

FEFCO 0427
Self locking box.

FEFCO 0713
Retail display packaging.

Each catalog entry should contain:

• FEFCO code
• illustration
• description
• typical use
• industries using it
